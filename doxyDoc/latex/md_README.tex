Welcome to the VS Code Java world. Here is a guideline to help you get started to write Java code in Visual Studio Code.\hypertarget{md_README_autotoc_md1}{}\doxysection{Folder Structure}\label{md_README_autotoc_md1}
The workspace contains two folders by default, where\+:


\begin{DoxyItemize}
\item {\ttfamily src}\+: the folder to maintain sources
\item {\ttfamily lib}\+: the folder to maintain dependencies
\end{DoxyItemize}

Meanwhile, the compiled output files will be generated in the {\ttfamily bin} folder by default.

\begin{quote}
If you want to customize the folder structure, open {\ttfamily .vscode/settings.\+json} and update the related settings there. \end{quote}
